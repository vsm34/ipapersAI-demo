Sample Research Paper Title

Author One, Author Two • Demo Conference 2025

Recent advances in machine learning have produced models that are able to perform complex language understanding tasks at scale. In this work we explore interactive tools for reading and annotating research papers, with an emphasis on workflows that help scholars track related work, surface key claims, and capture questions as structured notes.

We present a small demo application that supports inline highlighting, short comments, and a sidebar-driven note list that links back to highlights. Highlights are persisted locally and are associated with the page they appear on so users can contextually navigate longer documents.

The interface aims to mirror paper-based workflows: quick highlighters for important statements, color-coding for intent (e.g. question, highlight, idea), and a separate space for longer notes. Notes remain linked to their highlighted text so users can jump back to the exact passage.

Background and related work: there has been a long line of research on digital annotation systems, from early hypertext note-taking to modern collaborative document tools. Prior systems emphasize sharing, versioning, and search — our focus is on tight linking between short notes and highlighted passages for single-user sensemaking workflows.

We include sample content on this second page to demonstrate how annotations persist across pages and how the sidebar can be used to navigate long documents. Each highlight stores the page number and a short user note so contextual information is preserved.
